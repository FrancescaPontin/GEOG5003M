Internet User Classification - 2018 - Combined Authorities Geodata Pack: West Yorkshire (E47000003)


+ Abstract
This geodata pack provides the CDRC IUC for the year 2018 covering the Combined Authorities: West Yorkshire (E47000003)


+ Contents
	 - readme.txt: Information about the CDRC Geodata pack
	 - metadata.xml: Metadata
	 - tables: Folder containing the csv files
	 - shapefiles: Folder containing the shapefiles
